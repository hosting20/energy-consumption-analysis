# ⚡ Energy Consumption Analysis & Dashboard

## 📌 Project Overview
This project analyzes household electricity consumption data and provides an interactive dashboard for exploring usage patterns.

---

## 🎯 Objectives
- Analyze electricity consumption trends
- Identify peak usage hours
- Build a simple prediction model
- Create an interactive dashboard

---

## 📊 Dataset
- Source: UCI Machine Learning Repository
- Records: 2M+ rows
- Features: Date, Time, Global Active Power

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Streamlit

---

## 📈 Key Insights
- Peak consumption occurs in the evening
- Clear daily usage patterns
- Predictable trends over time
